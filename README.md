# Cyberpunk Character Generator

A Cyberpunk RED character generator built with React + TypeScript + Vite.

**Live:** _(deploy to Vercel)_

## Features

- Complete Package (point-buy) character creation — 10 Cyberpunk RED attributes, 62-point pool
- All 9 roles: Solo, Netrunner, Rockerboy, Tech, Medtech, Media, Exec, Lawman, Fixer
- Interactive stat editor with real-time validation and points counter
- Role abilities with starting rank 4
- Auto-calculated derived stats: HP, Humanity, Seriously Wounded, Death Save, Initiative
- Weighted random distribution biased toward role-appropriate stats
- Dark cyberpunk UI with glow effects, scanlines, and glitch animations

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build   # outputs to dist/
```
